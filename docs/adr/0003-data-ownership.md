# 0003 — Data ownership

**Дата:** 2025-11-19

**Рішення:** `catalog-service` володіє таблицями/схемами:
- `tracks`
- `artists`
- `albums`

Інші сервіси НЕ виконують cross-context JOIN на ці таблиці. Якщо потрібні дані — запит через API `catalog-service` або через асинхронну реплікацію (пізніше).

**Наслідки:** Обмеження JOIN знижує складність і залежності, але може збільшити кількість запитів між сервісами.
